# Working with Dates and Time

Almost every application deals with time — timestamps in logs, scheduling events, calculating deadlines, displaying "3 days ago." Python's `datetime` module handles it all.

## Core Concepts

```python
from datetime import datetime, date, time, timedelta

# Current date and time
now = datetime.now()          # 2024-01-15 14:30:45.123456
today = date.today()          # 2024-01-15

# Creating specific dates
birthday = date(2000, 6, 15)
meeting = datetime(2024, 3, 20, 14, 30)  # 2024-03-20 14:30:00
```

## Formatting Dates with strftime

`strftime` converts dates to formatted strings:

| Code | Meaning | Example |
|------|---------|---------|
| `%Y` | 4-digit year | `2024` |
| `%m` | Month (01-12) | `03` |
| `%d` | Day (01-31) | `20` |
| `%H` | Hour 24h (00-23) | `14` |
| `%I` | Hour 12h (01-12) | `02` |
| `%M` | Minute | `30` |
| `%S` | Second | `45` |
| `%p` | AM/PM | `PM` |
| `%A` | Day name | `Monday` |
| `%B` | Month name | `March` |

### Practice: Date Formatting

<interactive-code>
from datetime import datetime, date, timedelta

now = datetime.now()

# Different format styles
print(f"ISO format:    {now.strftime('%Y-%m-%d %H:%M:%S')}")
print(f"US style:      {now.strftime('%m/%d/%Y')}")
print(f"Human:         {now.strftime('%B %d, %Y')}")
print(f"With time:     {now.strftime('%I:%M %p on %A')}")

# Parsing a string into a date
date_str = "2024-12-25"
christmas = datetime.strptime(date_str, "%Y-%m-%d")
print(f"\nParsed: {christmas.strftime('%A, %B %d, %Y')}")
</interactive-code>

## timedelta: Date Math

`timedelta` represents a duration — the difference between two dates:

```python
from datetime import timedelta

today = date.today()
week_later = today + timedelta(weeks=1)
month_ago = today - timedelta(days=30)
```

### Practice: Date Calculations

<interactive-code>
from datetime import datetime, timedelta

now = datetime.now()

# Future and past dates
tomorrow = now + timedelta(days=1)
next_week = now + timedelta(weeks=1)
last_month = now - timedelta(days=30)

print(f"Now:        {now.strftime('%Y-%m-%d')}")
print(f"Tomorrow:   {tomorrow.strftime('%Y-%m-%d')}")
print(f"Next week:  {next_week.strftime('%Y-%m-%d')}")
print(f"Last month: {last_month.strftime('%Y-%m-%d')}")

# Calculate days between dates
new_years = datetime(now.year + 1, 1, 1)
days_left = (new_years - now).days
print(f"\nDays until New Year: {days_left}")

# Business day counter
def business_days(start, end):
    days = 0
    current = start
    while current < end:
        if current.weekday() < 5:  # Mon=0, Fri=4
            days += 1
        current += timedelta(days=1)
    return days

start = datetime(2024, 1, 1)
end = datetime(2024, 1, 31)
print(f"Business days in Jan 2024: {business_days(start, end)}")
</interactive-code>

> **Pro Tip:** Always store timestamps in UTC and convert to local time for display. This prevents timezone bugs when users are in different locations.

> **Common Mistake:** Comparing `datetime` objects with `date` objects — they're different types. Use `.date()` to extract the date from a datetime: `now.date() == date.today()`.
