# JSON: The Language of APIs

<DataStructuresVisualizer />

JSON (JavaScript Object Notation) is the standard format for sending and receiving data between applications. Every web API, configuration file, and data exchange you'll encounter uses JSON. Python's `json` module makes it seamless to work with.

## Core Operations

```python
import json

# Python → JSON string (serialization)
data = {"name": "Alice", "scores": [95, 88]}
json_str = json.dumps(data, indent=2)

# JSON string → Python (deserialization)
parsed = json.loads(json_str)

# Python → JSON file
with open("data.json", "w") as f:
    json.dump(data, f, indent=2)

# JSON file → Python
with open("data.json") as f:
    loaded = json.load(f)
```

### Practice: JSON Operations

<interactive-code>
import json

# API-like response data
api_response = {
    "status": "success",
    "data": {
        "users": [
            {"id": 1, "name": "Alice", "role": "admin", "active": True},
            {"id": 2, "name": "Bob", "role": "user", "active": False},
            {"id": 3, "name": "Charlie", "role": "moderator", "active": True},
        ],
        "total": 3
    },
    "metadata": {"page": 1, "per_page": 10}
}

# Serialize with pretty printing
json_text = json.dumps(api_response, indent=2)
print("JSON output:")
print(json_text[:200] + "...")

# Parse and extract data
parsed = json.loads(json_text)
active_users = [u["name"] for u in parsed["data"]["users"] if u["active"]]
print(f"\nActive users: {active_users}")

# Save and reload
with open("api_data.json", "w") as f:
    json.dump(api_response, f, indent=2)

with open("api_data.json") as f:
    reloaded = json.load(f)
print(f"Reloaded {reloaded['data']['total']} users")
</interactive-code>

## HTTP Requests (Concepts)

APIs communicate over HTTP. Here's how a typical request-response cycle works:

<MermaidDiagram>
graph LR
  CLIENT["Your Python Code"] -- HTTP Request --> SERVER["API Server"]
  SERVER -- JSON Response --> CLIENT
  style CLIENT fill:#3b82f6,stroke:#2563eb,color:#fff
  style SERVER fill:#22c55e,stroke:#16a34a,color:#fff
</MermaidDiagram>

| Method | Purpose | Example |
|--------|---------|---------|
| `GET` | Retrieve data | Get user profile |
| `POST` | Create new data | Create new user |
| `PUT` | Update existing data | Update user email |
| `DELETE` | Remove data | Delete a user |

```python
# With the requests library (pip install requests):
import requests

response = requests.get("https://api.example.com/users")
data = response.json()    # Auto-parses JSON response
```

> **Pro Tip:** Use `json.dumps(data, default=str)` to handle non-serializable types like `datetime` objects — it converts them to strings automatically.

> **Common Mistake:** Confusing `json.dump` (writes to file) with `json.dumps` (returns string). The `s` stands for "string."
