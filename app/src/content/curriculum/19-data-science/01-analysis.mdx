## NumPy: Fast Number Crunching

NumPy arrays are 50x faster than Python lists for numerical operations:

```python
import numpy as np

arr = np.array([1, 2, 3, 4, 5])
print(arr * 2)          # [2, 4, 6, 8, 10] — vectorized!
print(arr.mean())       # 3.0
print(arr.std())        # 1.414...
```

## Pandas: Data in Tables

import pandas as pd

df = pd.read_csv("data.csv")

### Practice: Data Analysis Simulation

# Simulating Pandas-style analysis without the library
class Series:
    def __init__(self, data, name=""):
        self.data = data
        self.name = name
    
    def mean(self): return sum(self.data) / len(self.data)
    def min(self): return min(self.data)
    def max(self): return max(self.data)
    def sum(self): return sum(self.data)
    
    def describe(self):
        n = len(self.data)
        sorted_d = sorted(self.data)
        return {
            "count": n,
            "mean": self.mean(),
            "min": self.min(),
            "max": self.max(),
            "median": sorted_d[n//2] if n % 2 else (sorted_d[n//2-1] + sorted_d[n//2]) / 2,
        }

# Sales data analysis
monthly_sales = Series(
    [12000, 15000, 11000, 18000, 22000, 19000, 
     25000, 23000, 20000, 17000, 21000, 28000],
    "Monthly Sales"
)

stats = monthly_sales.describe()
print("Sales Statistics:")
for key, value in stats.items():
    if isinstance(value, float):
        print(f"  {key}: ${value:,.0f}")
    else:
        print(f"  {key}: {value}")

# Find best and worst months
months = ["Jan","Feb","Mar","Apr","May","Jun","Jul","Aug","Sep","Oct","Nov","Dec"]
best_idx = monthly_sales.data.index(monthly_sales.max())
worst_idx = monthly_sales.data.index(monthly_sales.min())
print(f"\nBest month: {months[best_idx]} (${monthly_sales.max():,})")
print(f"Worst month: {months[worst_idx]} (${monthly_sales.min():,})")

# Growth rate
growth = [(monthly_sales.data[i] - monthly_sales.data[i-1]) / monthly_sales.data[i-1] * 100 
          for i in range(1, len(monthly_sales.data))]
print(f"\nAvg monthly growth: {sum(growth)/len(growth):.1f}%")

## The Workflow

1. **Load** — `pd.read_csv()`, `pd.read_json()`, `pd.read_sql()`
2. **Explore** — `.head()`, `.info()`, `.describe()`, `.shape`
3. **Clean** — `.dropna()`, `.fillna()`, `.rename()`, `.astype()`
4. **Transform** — `.groupby()`, `.merge()`, `.pivot_table()`
5. **Visualize** — `.plot()`, Matplotlib, Seaborn

> **Pro Tip:** Install Jupyter Notebook (`pip install jupyter`) for interactive data exploration. It's the standard tool for data science work.

> **Common Mistake:** Modifying a DataFrame slice without `.copy()`. Use `df_subset = df[df["age"] > 25].copy()` to avoid the `SettingWithCopyWarning`.
