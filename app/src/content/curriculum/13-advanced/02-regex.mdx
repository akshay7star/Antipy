# Regular Expressions (Regex)

Regular expressions are patterns that match text. They're used everywhere — form validation, log parsing, data extraction, search-and-replace. Python's `re` module provides full regex support.

## Basic Patterns

| Pattern | Matches | Example |
|---------|---------|---------|
| `.` | Any character | `c.t` → `cat`, `cut`, `c8t` |
| `\d` | Any digit | `\d\d` → `42`, `99` |
| `\w` | Word character (letter/digit/_) | `\w+` → `hello_world` |
| `\s` | Whitespace | `\s+` → spaces, tabs, newlines |
| `^` | Start of string | `^Hello` |
| `$` | End of string | `world$` |
| `*` | 0 or more | `ab*c` → `ac`, `abc`, `abbc` |
| `+` | 1 or more | `ab+c` → `abc`, `abbc` |
| `?` | 0 or 1 | `colou?r` → `color`, `colour` |
| `[]` | Character class | `[aeiou]` → any vowel |

## Core Functions

```python
import re

re.search(pattern, text)     # Find first match anywhere
re.match(pattern, text)      # Match at start only
re.findall(pattern, text)    # Find ALL matches
re.sub(pattern, repl, text)  # Find and replace
re.split(pattern, text)      # Split by pattern
```

### Practice: Regex Operations

<interactive-code>
import re

text = "Contact us at support@example.com or sales@company.org for info."

# Find all email addresses
emails = re.findall(r'\b[\w.+-]+@[\w-]+\.\w+\b', text)
print(f"Emails found: {emails}")

# Find all words starting with a capital letter
capitals = re.findall(r'\b[A-Z]\w*', text)
print(f"Capitalized: {capitals}")

# Replace phone format
phone_text = "Call 123-456-7890 or 987-654-3210"
formatted = re.sub(r'(\d{3})-(\d{3})-(\d{4})', r'(\1) \2-\3', phone_text)
print(f"\nFormatted: {formatted}")

# Validate a pattern
def is_valid_email(email):
    pattern = r'^[\w.+-]+@[\w-]+\.\w{2,}$'
    return bool(re.match(pattern, email))

test_emails = ["user@example.com", "bad@", "test@site.co.uk", "@nope.com"]
for email in test_emails:
    print(f"  {email}: {'valid' if is_valid_email(email) else 'invalid'}")
</interactive-code>

> **Pro Tip:** Use raw strings `r'...'` for regex patterns. Without the `r`, backslashes get interpreted by Python before regex sees them: `r'\d+'` is correct, `'\d+'` requires `'\\d+'`.

> **Common Mistake:** Using regex when simple string methods work. `"hello" in text` is faster and clearer than `re.search(r"hello", text)`. Use regex only when you need pattern matching.
